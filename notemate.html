AI-Powered Academic Platform

NoteMate

A smart academic collaboration platform that enables secure homework submission, online tests, real-time help requests, and efficient faculty-student interaction.

Project Abstract

NoteMate is an academic collaboration platform designed for colleges and institutions. It allows students to submit assignments, request help, and interact with peers, while giving faculty control over announcements, secure submissions, and feedback — all within a clean, user-friendly interface.

Project Objectives

  • To simplify the homework and note submission process through a centralized digital platform.
  • To help students request academic assistance and connect with peers and faculty for collaborative learning.
  • To enable faculty to post announcements and manage submissions efficiently.
  • To conduct secure online tests with time restrictions and anti-cheating measures.
  • To build a user-friendly interface that works across devices and screen sizes.
  • To lay the foundation for AI-powered tools like note summarization and learning progress tracking in future updates.

Key Features

  • Homework Submission: Students upload notes and assignments (text/files/images) with subject and timestamp.
  • Online Exam Module: Secure, timed tests with tab-switch detection and anti-copy-paste restrictions.
  • Request Help Section: Students can post doubts; responses allowed from faculty and peers.
  • Faculty Announcements: Highlighted posts from faculty with searchable interface.
  • Responsive UI: Mobile-friendly layout with offcanvas menu and sticky navbar.

Technologies Used

HTML Tailwind CSS JavaScript PHP MySQL OpenRouter API

Planned Enhancements

  • AI-based note summarization
  • Student learning analytics dashboard
  • Peer feedback and gamified ranking system
  • Career skill mapping based on submissions
