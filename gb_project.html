Grammar Base

An intelligent e-learning platform tailored for first-year engineering students to master English grammar with real-time AI tools and interactive features.

Project Abstract

Grammar Base is a dynamic, responsive web platform designed to make English grammar learning simple, engaging, and effective. Targeted especially for first-year engineering students, it offers interactive grammar topics, quizzes, and tools powered by AI to ensure conceptual clarity, practice, and fluency.

Project Objectives

  • To simplify complex English grammar concepts for first-year engineering students.
  • To make grammar learning engaging through quizzes, AI-based tools, and voice interaction.
  • To provide a dynamic platform with personalized dashboards and real-time feedback.
  • To offer admin-controlled study material uploads and secure content management.
  • To help students track progress and improve step-by-step using interactive tools.

Key Features

  • Login/Signup with Role-Based Access: Student and Admin roles with secured session management.
  • Interactive Grammar Topics: Prepositions, Adjectives, Pronouns, Tenses, and more – explained with images and examples.
  • Dynamic AI Quiz: Real-time question generation with difficulty levels and instant score feedback.
  • AI Voice Assistant: Voice-based interaction for grammar support and chatbot functionality.
  • Grammar & Spell Checker Tool: Real-time grammar correction powered by API.
  • Admin Upload Materials: Study resources uploadable only by Admin with dynamic access for students.
  • Progress Tracking Dashboard: Personalized learning stats and history of quiz results.
  • Responsive UI: Optimized for desktop, tablet, and mobile devices.

Technology Used

HTML CSS JavaScript Bootstrap PHP MySQL OpenRouter API PHPMailer

Highlights

  • Secure authentication with email verification.
  • Clean UI with AOS animations for smooth interaction.
  • AI-enabled interaction tools to enhance learning.
  • Scalable and customizable for other academic subjects.

Target Users

  • First-Year Engineering Students
  • Language Trainers and Instructors
  • Colleges and Universities
  • Open Learners & Peer Groups
