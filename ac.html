AC Service Website

A professional service website for an AC and home appliance repair company with dynamic gallery, contact form, and admin upload panel.

Project Abstract

This AC Service Website is designed for a home appliance service company that provides repair and maintenance for air conditioners, washing machines, fridges, RO water systems, and electrical services. It allows users to view services, browse a dynamic image/video gallery, and contact the company. An admin panel is included for the business owner to upload and manage gallery content securely.

Project Objectives

  • To build a modern, responsive website for a local AC/home service company.
  • To allow customers to view services and easily contact the company.
  • To provide a clean image/video gallery of completed works.
  • To enable the owner to upload or delete gallery content securely via a login panel.
  • To implement the site using technologies supported by GoDaddy (PHP, MySQL).

Key Features

  • Responsive Design: Tailwind CSS-based modern layout, mobile friendly.
  • Service Showcase: Pages for AC, fridge, washing machine, RO, and electrical work.
  • Contact Page: With direct WhatsApp integration and email form.
  • Gallery Module: Separate image/video sections with owner-only upload access.
  • Admin Panel: Hidden login for owner to manage gallery securely.

Technologies Used

HTML Tailwind CSS JavaScript PHP MySQL
